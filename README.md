# NEWordRemberer

一个基于 Web 技术开发的英语单词背诵应用，采用记忆曲线算法帮助用户高效记忆单词。

## 功能特点

- **单词管理**：支持单词搜索、添加，包含词性和中文释义
- **背诵任务**：自定义每日背诵数量和复习数量
- **两种模式**：中译英（输入单词）和英译中（选择认识/不熟/不认识）
- **记忆曲线**：采用简化的 SM-2 算法，根据记忆结果动态调整复习间隔
- **错题重开**：支持多次错题重开，每次只重开上次未背出的单词
- **重新背诵**：支持重新背诵今日计划或首次错题重测
- **数据存储**：使用浏览器 localStorage 本地存储
- **移动端适配**：响应式设计，支持手机端使用

## 快捷键

在背诵页面支持以下快捷键：

| 按键 | 功能 |
|------|------|
| Enter | 提交答案或进入下一个单词 |
| 0 | 标记为"认识" |
| 8 | 标记为"不熟" |
| 9 | 标记为"不认识" |

## 使用方法

1. 直接打开 `index.html` 文件即可使用
2. 首次使用会自动加载默认词库
3. 在首页创建背诵任务，设置新单词和复习单词数量
4. 点击"开始背诵"进入背诵界面
5. 完成背诵后可查看结果，支持错题重开

## 项目结构

```

├── index.html              # 主页面
├── style.css               # 样式文件
├── app.js                  # 应用入口
├── data/
│   └── defaultWords.js     # 默认词库（313个单词）
└── modules/
    ├── WordBank.js         # 词库管理模块
    ├── TaskManager.js      # 任务管理模块
    ├── MemoryCurve.js      # 记忆曲线算法模块
    └── UIManager.js        # UI管理模块
```
